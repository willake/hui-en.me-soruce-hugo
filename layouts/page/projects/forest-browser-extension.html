Introduction
Forest Browser Extension is a mini gamification tool on Google Chrome and Firefox. It is like Forest but a function-limited version. In this extension, users could set up a timer and it will blocked all website except those in the allowed list. It was my first project in SeekrTech. I refactored the whole code base and developed new features that connect with Forest server. The experience was very valuable since I collaborated with a professional backend engineer and learnt a lot of knowledge about networking and database design.
